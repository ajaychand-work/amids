CittaAI Phase 1 Beta MVP

This project is now a runnable full-stack MVP.

Start it:

  1. cd cittaai_phase1_beta
  2. python -m venv .venv
  3. .\.venv\Scripts\Activate.ps1
  4. pip install -r requirements.txt
  5. uvicorn backend.app.main:app --reload --port 8000

Then open http://127.0.0.1:8000.

Frontend source: web/index.html
Backend source: backend/app/main.py
